<h2 style="color: darkgray;">Guide étape 1 - Installation du projet</h2>

```bash
    sudo apt-get install git-flow
```
```bash
    symfony new web_indus 3.4
```
```bash
    cd web_indus
```
```bash
    composer install
```
```bash
    git flow init
```
```bash
    sf3 cache:clear
```
```bash
    sf3 server:run
```
